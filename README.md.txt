🧠 Inventory Management System (Python + Tkinter)

🔍 Overview
This is a beginner-friendly Python project to manage product inventory using GUI, login system, and reporting.

✅ Features
- 🔐 User login & authentication
- 📦 Add / Edit / Delete products
- 📉 Low stock alerts (Qty < 5)
- 📊 Export inventory as CSV
- 💡 GUI made using Tkinter and ttk


🚀 Getting Started

1. Install Python (>=3.6)
2. Clone the repo:
```bash
git clone https://github.com/your_StackInSync/Inventory-Management-System.git
cd Inventory-Management-System
